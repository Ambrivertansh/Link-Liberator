# Link-Liberator
Link-Liberator is a headless browser automation tool built to navigate through URL shorteners that require user interaction (timers, clicks, and captchas). Unlike simple HTTP resolvers, this project simulates a real user environment to bypass client-side logic.

Built for: Python 3 & Selenium

Optimized for: Google Colab (Run it from your phone via the cloud)

Features: Handles redirects, manages pop-ups, and extracts final destination URLs.
